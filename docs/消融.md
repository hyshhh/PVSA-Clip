# 消融实验命令

## 骨干 CNN Block 消融

支持的 `cnn_block_type`：

```python
'dwconv'
'dwconv_act'
'mbconv'
'mbconv_no_se'
'c2f'
'c3k2'
'convnext'
```

跑全部骨干块消融：

```bash
CUDA_VISIBLE_DEVICES=0 python tools/analysis_tools/run_backbone_block_ablation.py \
  configs-h/biformer/biformer_baseline_camvid.py \
  --work-dir-root work_dirs/backbone_block_ablation_camvid \
  --shape 256 256 \
  --skip-existing
```

只跑指定几组：

```bash
CUDA_VISIBLE_DEVICES=0 python tools/analysis_tools/run_backbone_block_ablation.py \
  configs-h/biformer/biformer_baseline_camvid.py \
  --work-dir-root work_dirs/backbone_block_ablation_camvid \
  --cnn-block-types c2f c3k2 convnext \
  --shape 256 256 \
  --skip-existing
```

只重新汇总已有结果：

```bash
python tools/analysis_tools/run_backbone_block_ablation.py \
  configs-h/biformer/biformer_baseline_camvid.py \
  --work-dir-root work_dirs/backbone_block_ablation_camvid \
  --summary-only
```

汇总文件：

```text
work_dirs/backbone_block_ablation_camvid/summary.csv
```

汇总字段：

```text
run_name,cnn_block_type,flops,params,best_mIoU,status
```
